Chapter 1
Introduction & Overview

1.1 About the Website

This website was custom-built for House of Israel Assembly, a Hebrew community situated at Burma Barracks Chapel, Lusaka, Zambia, under the leadership of Rabbi Dr. Phillip Banda.

The website serves as the online home for the community, enabling visitors to learn about the assembly, watch sermons, register for events, view photos, submit prayer requests, and make donations — all through a modern, professionally designed interface.

7 Public Web Pages

Home, About, Sermons, Events, Gallery, Contact, and Give — each fully designed.

Full Admin CMS Panel

A private dashboard for managing all website content without touching code.

Dynamic Animations

Floating Hebrew letter particles, rotating Star of David, and scroll-triggered animations.

Sermon & Event Management

Add, edit, and delete sermons and events directly from the admin dashboard.

Prayer Request System

Visitors submit prayer requests; the admin team can view and mark them as prayed.

Fully Responsive Design

Looks excellent on mobile phones, tablets, and desktop computers.

1.2 Technology Stack

ComponentTechnologyPurpose
Backend ServerNode.js + Express.jsRuns the website and handles all API requests
DatabaseJSON FilesStores sermons, events, prayers, content
AuthenticationJWT (JSON Web Tokens)Secures the admin panel login
Frontend DesignHTML5, CSS3, JavaScriptThe visual website the public sees
AnimationsAOS Library + Canvas APIScroll animations & floating particles
IconsFont Awesome 6All icons throughout the website
File UploadsMulterHandles gallery image uploads

1.3 System Requirements

To run the website on your computer, you need the following:

  • Node.js — Version 18 or higher (installed at https://nodejs.org)
  • Operating System — Windows 10/11, macOS, or Ubuntu Linux
  • RAM — Minimum 512 MB available
  • Storage — Minimum 200 MB free space
  • Internet Browser — Chrome, Edge, Firefox, or Safari (modern version)
  • Internet Connection — Required only to load fonts and icons from CDN
ℹ️ Node.js is already installed
If you are running this website on the same computer where it was set up, Node.js v24 is already installed and the dependencies are ready. Simply run START.bat to launch the server.

Chapter 2
Getting Started

2.1 Starting the Website Server

The website runs on a local server. You must start the server every time you want to use the website. Follow these steps:

1
Open File Explorer Navigate to the folder C:\HouseOfIsrael\ on your computer.
2
Double-click START.bat This file starts the website server automatically. A black Command Prompt window will open.
3
Wait for the confirmation message You will see a message saying the server is running at http://localhost:3000. This means the website is ready.
4
Keep the window open Do not close the Command Prompt window while using the website. Closing it will stop the server.
⚠️ Important
The Command Prompt (black window) must remain open for the website to work. You can minimise it but do not close it.

2.2 Accessing the Website & Admin Panel

Once the server is running, open your web browser and go to these addresses:

Admin Loginhttp://localhost:3000/adminAdministrator only

Chapter 3
Public Website Pages

The website has 7 public pages that visitors can see. Each page has a consistent design with the House of Israel gold-and-dark colour theme, Hebrew decorative elements, and smooth animations.

3.1 Homepage index.html

The homepage is the main entry point for all visitors. It contains the following sections:

  • Hero Section — Full-screen animated background with floating Hebrew letters (✡ א ב ג), the assembly name in large gold text, tagline, and location
  • Announcement Bar — Gold banner showing service times
  • About Preview — Brief description with animated statistics (members, services, sermons)
  • Rabbi Quote — A featured quote from Rabbi Dr. Phillip Banda
  • Services Grid — Cards showing Shabbat, Torah Study, Prayer Meeting, Youth Program
  • Latest Sermons — The 3 most recent sermons (auto-loaded)
  • Upcoming Events — Next 3 events (auto-loaded)
  • Prayer Request Form — Visitors can submit prayer requests
  • Gallery Preview — Photo grid teaser
  • Location & Contact — Address, phone, email, social links
  • Newsletter Signup — Email subscription form
  • Footer — Links, service times, Hebrew Shema text

3.2 About Page about.html

Introduces the assembly to new visitors. Contains:

  • Mission statement and community description
  • Our Beliefs — 6 belief cards covering Torah, Shabbat, feasts, community, and Hebrew roots
  • Rabbi Profile — Full profile of Rabbi Dr. Phillip Banda with credentials
  • History Timeline — The journey of the assembly from founding to today
  • A scripture callout (Jeremiah 6:16)

3.3 Sermons Page sermons.html

A searchable, filterable library of all teachings. Features include:

  • Search Bar — Search by sermon title, speaker, or description
  • Category Filters — All, Torah Portion, Shabbat Teaching, Special Teaching
  • Sermon Cards — Each card shows the title, date, speaker, duration, category, and a Watch Now button
  • YouTube links open directly in YouTube when provided by the admin

3.4 Events Page events.html

Shows all upcoming community gatherings. Contains:

  • 2025 Biblical Feast Calendar — A visual grid showing all 6 major feasts with Hebrew names and dates
  • Events List — All events sorted by date with date box, description, category, and action button
  • Event Registration Form — Visitors can fill in their name, email, phone, number of guests, and select the event they wish to attend

3.5 Gallery Page gallery.html

A photo showcase of community life. Features:

  • Category Filter Buttons — All Photos, Services, Torah Study, Events, Community, Youth
  • Masonry-style grid layout — Photos display in an attractive variable-size grid
  • Lightbox viewer — Clicking any image opens it in a full-screen overlay
  • Images are uploaded through the Admin Panel (see Chapter 4.6)

3.6 Contact Page contact.html

Provides all ways to reach the assembly. Includes:

  • Contact information (address, phone, email, service times)
  • Contact Form — Name, email, phone, subject dropdown, message
  • Social media buttons (Facebook, YouTube, WhatsApp, Instagram, TikTok)
  • FAQ Section — Expandable answers to common questions about Hebrew community

3.7 Give Page give.html

The online giving and donations page. Contains:

  • Quick Amount Selector — Preset amounts (K50, K100, K200, K500, K1000) plus custom
  • Giving Type — Tithe, First Fruits, Freewill, Feast Offering, Building Fund, Media Ministry, Youth
  • Payment Methods — Mobile Money (Airtel/MTN), Bank Transfer, In-Person
  • Impact Section — 6 cards showing where donations go
  • Scripture callout (Malachi 3:10)
💡 Customising Pages
The main text content on these pages (hero text, about text, rabbi quote, contact info) can be updated through the Admin Panel → Edit Homepage. For structural changes, the HTML files in C:\HouseOfIsrael\public\ can be edited with any text editor.

Admin Panel Guide

The Admin Panel is a private, password-protected control centre that allows you to manage all website content without needing any technical knowledge. Access it at http://localhost:3000/admin

4.1 Logging In

1
Open your browser and go to http://localhost:3000/admin
2
Enter your username — default is admin
3
Enter your password — default is hoisrael2025
4
Click "Sign In" — You will be taken to the dashboard
⚠️ Security Reminder
Change the default password immediately after first use. Go to Settings → Change Password in the admin panel. Do not share your login credentials with anyone outside the authorised team.

4.2 Dashboard Overview

The dashboard is the first page you see after logging in. It shows:

  • Statistics Cards — Total sermons, events, prayer requests, and newsletter subscribers at a glance
  • Quick Actions — Shortcut buttons: Add Sermon, Add Event, Upload Image, View Prayers, Edit Homepage, View Subscribers
  • Left Sidebar — Navigation to all admin sections
  • Live Clock — Current time shown in the top bar

4.3 Editing the Homepage Content

Go to Edit Homepage in the sidebar. You can change the following:

SectionWhat you can edit
Hero SectionBadge text, title (lines 1 & 2), subtitle/tagline, location text
About SectionTwo description paragraphs, member count, services per year, online sermons count
Rabbi QuoteThe featured quote text, author name, and author title
Contact InformationAddress, phone number, email, Facebook, YouTube, WhatsApp, Instagram URLs
1
Click "Edit Homepage" in the left sidebar
2
Update the text fields you want to change
3
Scroll down and click "Save All Changes" — Changes apply instantly to the live website

4.4 Managing Sermons

Go to Manage Sermons in the sidebar to see all sermons in a list.

Adding a New Sermon

1
Click "Add New Sermon" button at the top right
2
Fill in the sermon details:
  • Sermon Title — The full title of the teaching
  • Category — Torah Portion, Shabbat Teaching, Special Teaching, etc.
  • Date — The date the sermon was delivered
  • Speaker — Defaults to "Rabbi Dr. Phillip Banda"
  • Duration — e.g. "58 min" or "1 hr 20 min"
  • YouTube Link — Paste the YouTube URL if the sermon is on YouTube
  • Description — A brief summary of the teaching
  • Featured — Tick this to mark as a featured sermon
3
Click "Save Sermon" — It will appear on the Sermons page immediately

Editing or Deleting a Sermon

Each sermon row has two buttons: Edit (pencil icon) to update it, and Delete (trash icon) to remove it permanently. Deletion is instant and cannot be undone.

4.5 Managing Events

Go to Manage Events in the sidebar to see all events.

Adding a New Event

1
Click "Add New Event" button
2
Fill in the event details:
  • Event Title — Name of the event (e.g. "Rosh Hashanah Celebration")
  • Date — Select the date from the calendar picker
  • Time — e.g. "9:00 AM" or "Sundown"
  • Location — Defaults to "Burma Barracks Chapel"
  • Category — Feast Day, Conference, Workshop, Youth Event, etc.
  • Description — Details about the event
  • Registration Required — Tick if people need to register in advance
3
Click "Save Event" — It appears on the Events page immediately, sorted by date

Editing or Deleting an Event

Use the Edit button to modify event details, or Delete to permanently remove it. Events in the past are automatically moved down in the list.

4.6 Gallery Management

Go to Manage Gallery in the sidebar. You can upload photos from community services, events, and activities.

Uploading a New Image

1
Click "Select Image" in the upload area
2
Choose a photo from your computer (JPG, PNG, WEBP supported; max 5 MB per image)
3
A preview will appear. Add a Caption (e.g. "Shabbat Service June 2025") and a Category (e.g. Service, Youth, Events)
4
Tick "Featured (large)" if you want this image to appear larger in the gallery grid
5
Click "Upload" — The image is saved and appears on the Gallery page immediately
💡 Image Tips
  • Use landscape photos (wider than tall) for best results
  • Recommended minimum size: 800×600 pixels
  • Good photos to upload: Shabbat services, Torah study sessions, feast celebrations, youth activities, community meals

4.7 Prayer Requests

Go to Prayer Requests in the sidebar to see all requests submitted by visitors.

Each request shows the submitter's name (or "Anonymous"), email (if provided), the date submitted, and the full prayer request text.

Click Mark Prayed next to a request to record that your team has prayed for it. This updates the status and the button changes to a green "Prayed" tag.

4.8 Newsletter Subscribers

Go to Newsletter in the sidebar to see a list of all email addresses that have subscribed to your newsletter. Each entry shows the email address and the date they subscribed.

You can use this list to send out newsletters using any email service (such as Gmail, Mailchimp, or any other tool).

Chapter 5
Deployment Guide

Currently the website runs only on your local computer (localhost). To make it accessible on the internet — so that anyone anywhere can visit it — you need to deploy it to a hosting server. Below are three options, from easiest to most advanced.

5.1 Option A: Local Network Sharing Simplest

Share the website with people on the same Wi-Fi or local network (e.g. within the church building).

1
Find your computer's IP address — Open Command Prompt and type ipconfig. Look for the number next to "IPv4 Address" (e.g. 192.168.1.5)
2
Start the server with START.bat as usual
3
Others on the same Wi-Fi can visit http://192.168.1.5:3000 (replace with your actual IP)
ℹ️ Limitation
This only works while your computer is on and connected to the same network. It is not accessible from outside your home or church building.

5.2 Option B: Free Cloud Hosting (Railway or Render) Recommended

This makes the website permanently accessible on the internet for free. Railway.app is the easiest platform for this type of Node.js website.

Step 1 — Push your code to GitHub

1
Create a free account at https://github.com
2
Create a new repository named house-of-israel-website and make it Private
3
Install Git from https://git-scm.com if not already installed
4
Open Command Prompt in C:\HouseOfIsrael and run:
git init git add . git commit -m "Initial website" git remote add origin YOUR_GITHUB_URL git push -u origin main
Note: Before pushing, create a .gitignore file with node_modules/ and uploads/ on separate lines.

Step 2 — Deploy on Railway.app

1
Go to https://railway.app and sign up with your GitHub account
2
Click "New Project" → Deploy from GitHub Repo" and select your repository
3
Railway auto-detects Node.js. In the Variables tab, add: PORT = 3000 and JWT_SECRET = your-secret-key-here
4
Click Deploy. In 2–3 minutes, Railway gives you a free URL like https://house-of-israel.railway.app
5
Share this URL with anyone in the world — they can access the website from any device
✅ Railway Free Tier Includes
500 hours/month of uptime, 1 GB RAM, 1 GB storage. This is more than enough for a community website.

5.3 Option C: VPS Server Deployment Advanced

A Virtual Private Server (VPS) gives you the most control. Recommended providers: DigitalOcean ($6/month), Hetzner ($4/month), or Contabo.

1
Create a Ubuntu 22.04 server (1 GB RAM is sufficient)
2
SSH into the server and install Node.js:
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash - sudo apt-get install -y nodejs
3
Upload your website files (using FileZilla SFTP or git) and run npm install
4
Install PM2 to keep the server running 24/7:
npm install -g pm2 pm2 start server.js --name "house-of-israel" pm2 startup pm2 save
5
Install Nginx as a reverse proxy and point it to port 3000
6
Get a free SSL certificate with Let's Encrypt: sudo certbot --nginx -d yourdomain.com

5.4 Registering a Domain Name

A domain name (e.g. www.houseofisraelassembly.org) makes your website professional and easy to remember.

RegistrarApproximate CostNotes

5.5 SSL / HTTPS Setup

HTTPS encrypts the connection between your website and visitors. It is required for any public website today. Both Railway and Render automatically provide free SSL certificates. For VPS, use Let's Encrypt (free) with the certbot tool.

💡 Recommendation
For a first-time deployment, use Railway.app (Option B). It is free, automatic, and requires no server management knowledge. You can always move to a VPS later.

Chapter 7
Troubleshooting
ProblemLikely CauseSolution
"Site can't be reached" in browser Server is not running Double-click START.bat. Look for the confirmation message in the black window.
Admin login says "Invalid credentials" Wrong username or password Username is admin (all lowercase). If password was changed and forgotten, contact your web developer to reset it.
START.bat closes immediately Node.js not installed or a code error Right-click START.bat → "Run as administrator". Or open Command Prompt, navigate to the folder, and type node server.js to see the full error message.
Sermons / Events not loading on public site Server is not running or browser cache Ensure START.bat is running. Press Ctrl+Shift+R to force-refresh the page in your browser.
Image upload fails in gallery File is too large or wrong format Images must be under 5 MB. Accepted formats: JPG, PNG, WEBP. Resize the image before uploading.
Website looks broken / no styles Files in the wrong location or browser cache Ensure files are in C:\HouseOfIsrael\public\css\. Hard refresh with Ctrl+Shift+R.
Admin panel shows "Token expired" You were logged in for more than 24 hours This is normal security behaviour. Simply log in again.
Port 3000 already in use Another application is using port 3000 Open Task Manager → End Task on any "node.exe" processes. Then restart START.bat.
Animations not working No internet connection (CDN libraries not loading) The website requires internet access to load animation libraries (AOS, Font Awesome) from the web. Check your internet connection.

Getting Technical Support

If you encounter an issue not listed above, contact your web developer with:

  1. A description of what you were trying to do
  2. The exact error message (take a screenshot if possible)
  3. The content of the black Command Prompt window (if it shows errors)
ℹ️ Error Logs
If the server shows errors in the Command Prompt window, copy the text (right-click → Select All → Enter) and send it to your developer. This helps diagnose problems quickly.
